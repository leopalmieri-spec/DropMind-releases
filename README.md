# DropMind — Releases

This repository hosts the binary releases and assets for **DropMind**, a desktop software for AliExpress → eBay dropshipping automation.

## 📦 What's here

- **Software releases** — Latest stable Windows builds of `DropMind.exe`
- **Category embeddings** — Pre-computed eBay category data for 20 countries (`categories-v1` tag)
